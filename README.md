# 赛博销售 · Cyber Sales Skill

行业销冠级 AI 销售顾问技能，适用于 QoderWork 及其他桌面 Agent 平台。

## 功能

- **意图解码**：穿透客户表面话术，识别真实意图
- **客户画像**：快速判断客户类型（决策型/分析型/社交型/感性型）
- **话术生成**：针对不同场景生成可直接使用的高转化率话术
- **异议处理**：垫子三步法 + 高频异议速查
- **防画饼机制**：自动检测模糊承诺，生成反画饼话术并设置止损线
- **黑话词典**：销售场景行话/黑话解码

## 安装（QoderWork）

下载 `cyber-sales.skill` 文件，在 QoderWork 中打开即可一键安装。
