## 一、RAIN 销售模式

### 框架总览

RAIN 是一套系统化的顾问式销售方法论，由 RAIN Group 创始人 Mike Schultz 和 John E. Doerr 提出，核心分为三大模块：

| 模块 | 核心内容 |
|------|---------|
| **准备篇** | 心态管理、自我对话、目标规划、价值定位 |
| **理论篇** | 寒暄建立好感、渴望与痛处挖掘、冲击力制造、新现实描绘、说服与询问平衡 |
| **实践篇** | 5个为什么深挖、影响力16原理、会谈8部曲、异议处理、签单技巧 |

### RAIN 核心原则

1. **Relax（放松）**：不要急于推销，先建立轻松氛围
2. **Ask（提问）**：用开放式问题引导客户自己发现需求
3. **Impact（冲击）**：用数据和情感双重冲击，让客户感受到问题的严重性
4. **New Reality（新现实）**：描绘合作后的美好未来，制造"前后对比"

### 自我对话管理

销售员的成败取决于内在自我对话：
- 面对拒绝时：普通销售员想"我不行"，顶尖销售员想"这次拒绝让我更接近成交"
- 面对困难客户时：不要想"这个客户太难搞"，要想"搞定这个客户我的能力会提升一个层级"
- 每日自我暗示："我感觉自己好棒，好棒，好棒"——重复三次以上，释放内啡肽

### 价值定位与电梯演讲

30秒电梯演讲公式：
```
[你的产品/服务] + [解决的核心痛点] + [带来的具体价值] + [差异化优势]
```

示例：「我们帮中小电商在30天内把退货率降低40%，目前行业里只有我们提供按效果付费的模式」

### 寒暄的艺术

- **乔·吉拉德250法则**：每个人背后有约250人的社交圈，每一次寒暄都可能带来连锁效应
- 寒暄目的：建立好感与信任，而非急于推销
- 不要为了把握机会而无限延长会谈——满脸倦怠的客户记不住你说的所有内容
- 在条件相当的供应商中，客户往往只因喜欢谁的销售而决定采购

### 渴望与痛处

- 挖掘客户的"渴望"（Desires）和"痛处"（Pain Points）
- 面对不爱讲话的客户：用开放式问题 + 行业案例撬开客户的嘴
- 深层需求往往被客户自己忽略，需要通过追问让其浮现

### 冲击力制造

双重冲击策略：
- **理性冲击**：量化ROI、节省成本、提升效率的具体数字
- **感性冲击**：自我满足感、同事认可、职业成就感、面子问题
- 两者结合效果最佳：「这不仅能帮你省下20万成本，更重要的是，老板会看到你在降本增效上的贡献」

### 新现实描绘

- 给客户描绘合作后的"新现实"——让他们看到改变后的样子
- 使用 Before/After 对比：「现在你们每月花3天做报表，用了我们的系统后，3分钟出结果」
- 让客户自己想象，而不是你替他说

### 5个为什么深挖法

源自丰田大野耐一，连续追问5层"为什么"找到根本原因：
```
客户：我们系统太慢了
→ 为什么慢？→ 数据量太大
→ 为什么数据量大就有问题？→ 数据库没做分表
→ 为什么没做分表？→ 当初开发时没想到会增长这么快
→ 为什么没想到？→ 当时没有做容量规划
→ 为什么没做容量规划？→ 没有专业的架构师
→ 根本需求：需要一个能持续做架构优化的技术团队
```

### 销售影响力16条原理

1. 互惠原理：先给予再索取
2. 承诺一致：小承诺引向大决定
3. 社会认同：从众效应
4. 喜好原理：人们从喜欢的人那里买
5. 权威效应：专业形象增加说服力
6. 稀缺原理：越难得到的越想要
7. 对比效应：先展示高价再展示目标价
8. 锚定效应：第一个数字影响后续判断
9. 损失厌恶：失去的痛苦 > 得到的快乐
10. 禀赋效应：拥有的东西会被高估价值
11. 框架效应：同样的信息不同表述产生不同效果
12. 光环效应：一个优点照亮全部
13. 互惠让步：你先让步对方也会让步
14. 喜好关联：与客户有共同点更容易成交
15. 权威转移：专家/权威人士的背书
16. 即时满足：人们倾向即时满足而非延迟满足

### 会谈主导8部曲

1. 开场寒暄（建立好感）
2. 议程设定（让客户知道接下来会怎样）
3. 现状了解（SPIN中的S和P）
4. 痛点深挖（SPIN中的I和N）
5. 方案呈现（针对性的解决方案）
6. 异议处理（垫子三步法）
7. 试探成交（假设成交/选择成交/紧迫成交）
8. 下一步约定（锁定具体时间，不说"到时候再说"）

### 异议处理深度指南

**区分真实异议与虚假异议**：
- 真实异议：具体、有细节、重复出现 → 需要认真解决
- 虚假异议：模糊、笼统、像借口 → 背后有真正没说出口的原因

**价格异议的5种真实含义**：
1. "我真的买不起" → 预算确实不够 → 提供分期/降级方案
2. "我想看看还有没有更便宜的" → 在比较 → 强调差异化价值
3. "习惯砍价" → 不砍白不砍 → 坚守价格，给小赠品
4. "在测试你的底线" → 想确认自己没吃亏 → 给一个"台阶下"的理由
5. "需要向领导交代" → 需要价格理由去汇报 → 提供ROI分析材料

**"让我考虑考虑"的破解**：
- "完全理解，您主要是在考虑哪方面呢？是效果、价格还是其他？"
- "关于这个产品，此刻你已经了解了全部信息。从您的情况来看，这似乎是个好选择，为什么不现在就定下来呢？"

## 二、SPIN 顾问式销售

### 框架总览

SPIN 是 Neil Rackham 在《SPIN Selling》中提出的大客户销售方法论，适用于长周期、高金额、多决策者的复杂销售。

### 四大提问类型

| 类型 | 英文 | 目的 | 示例 |
|------|------|------|------|
| S - 背景问题 | Situation | 了解客户现状 | "目前贵司的CRM系统用了多久？" |
| P - 难点问题 | Problem | 发现客户痛点 | "使用过程中遇到过哪些不便？" |
| I - 暗示问题 | Implication | 放大痛点影响 | "如果这个问题持续半年，对业绩影响有多大？" |
| N - 需求效益 | Need-Payoff | 让客户自己说出价值 | "如果响应速度提升50%，对客户满意度意味着什么？" |

### SPIN 使用原则

1. **背景问题不要太多** — 提前做功课，不要让客户做你的"功课老师"
2. **难点问题是核心** — 找到痛点才能展开后续
3. **暗示问题要具体** — 用数字量化影响，让客户"肉疼"
4. **需求效益让客户说** — 不要替客户说好处，让他自己说更有说服力

### 适用判断标准（5个问题）

1. 涉及多少个决策者？→ 越多越需要SPIN
2. 决策周期长还是短？→ 越长越需要
3. 决策在哪里做出？→ 不在你面前做出时更需要
4. 决策失误风险多大？→ 越大越需要
5. 购买金额大不大？→ 越大越需要

## 三、FAB 产品介绍法

### 框架

| 层次 | 含义 | 话术模式 |
|------|------|---------|
| F - Feature（特征） | 产品有什么？ | "我们的产品具备XX功能" |
| A - Advantage（优点） | 比别人好在哪？ | "相比传统方案，可以XX" |
| B - Benefit（利益） | 对客户意味着什么？ | "这意味着您每个月可以省下XX" |

### 使用原则

- 永远以B（利益）结尾，让客户聚焦在"我能得到什么"
- F和A是铺垫，B才是成交的关键
- 不同客户类型，B的侧重点不同：决策型重ROI，分析型重数据，社交型重口碑，感性型重体验

## 四、影响力六大原则（西奥迪尼）

### 1. 互惠原理
- 先给予再索取：免费试用、行业报告、专业咨询
- 小恩惠能换来大回报：一杯咖啡可能换来一个合同
- 给予要真诚、有价值，不能是廉价的"糖衣炮弹"

### 2. 承诺与一致
- 让客户做出小承诺 → 逐步引导至大决定
- "您觉得这个功能对您有帮助吗？"→ 小认同 → "那您觉得整体方案呢？"→ 大认同
- 人一旦公开表态，就有保持一致的心理压力

### 3. 社会认同
- 85%的销售发生在有人说产品好之后
- 展示同行/同类客户的选择："你们行业TOP10中有7家在用"
- 推荐信/案例是最强的销售武器

### 4. 喜好原理
- 人们从喜欢的人那里购买
- 建立喜好的方法：相似性、赞美、合作、关联
- 找到与客户的共同点：同乡、同校、同爱好

### 5. 权威效应
- 专业形象 = 值得信赖 = 产品靠谱
- 展示资质、奖项、专家背书
- 穿着得体：第一印象95%取决于外表

### 6. 稀缺原理
- 越难得到的越想要
- 限量、限时、限资格：「仅限本月」「仅限前50名」「需要申请资格」
- 必须真实——虚假稀缺一旦被识破，信任归零

## 五、客户购买心理学

### 购买决策的本质

**核心发现：购买100%是情感驱动**
- 客户先有情感冲动，再用逻辑合理化
- 销售 = 情感启动 + 逻辑验证

### 两大购买动因

| 动因 | 动力系数 | 应用 |
|------|---------|------|
| 希望获益 | 1.0x | "用了之后能提升XX%" |
| 害怕损失 | 2.5x | "不用的话每月多损失XX" |

→ 损失框架比获益框架有效2.5倍

### 11大购买需求（按驱动力排序）

1. 金钱 — 想拥有更多/节省更多
2. 安全 — 财务/情感/身体的安全感
3. 讨人喜欢 — 被周边人接受和尊重
4. 地位和名望 — 感到自己重要
5. 健康 — 长寿、精力充沛
6. 赞美和认可 — 成就被人看见
7. 权力和影响力 — 对周围有控制感
8. 领先潮流 — 成为第一批拥有者
9. 爱和友谊 — 建立深度人际连接
10. 个人发展 — 获得新知识和技能
11. 自我转变 — 变成更好的自己（最强驱动力）

### 购买决定的"满意单位"

客户每次购买都想获得尽可能多的"满意单位"——身体、情感、精神层面的满足。产品能在越多方面取悦客户，越容易成交。

### 暗示的力量

- 55%信息来自身体语言，38%来自声音，7%来自语言
- 沉着、自信、放松的销售人员具有极强的暗示影响力
- 环境暗示：专业的办公环境/展示厅能大幅提升成交率

### 弗洛伊德口误

- 让客户自由谈论，最终会"说漏嘴"——脱口而出真实想法
- 创造舒适环境，通过提问和倾听引出真实需求
- 不要打断客户的"口误"，顺着追问

### 购买阻抗

- 客户对推销有天然反感
- 对说服有天然抵抗
- 对保证有天然怀疑
- 解决方式：做顾问而非推销员，提供价值而非施加压力

## 六、读心术在销售中的应用

### 表情解读

**愉快表情**：嘴角向后咧 + 脸蛋上移 + 眼睛变细
**不愉快表情**：口角下垂 + 脸蛋下垂变细长 + 眉皱成"八"字

### 情感能量泄露三通道

当客户刻意隐藏情绪时，能量会从三个通道泄露：

1. **面部僵硬/痉挛** — 强装淡定但面部肌肉不自然
2. **手足无意识动作** — 频繁变换姿势、摆弄物品、脚抖
3. **语调/语速变化** — 突然变慢/变快、声音变尖/变低

### 识破虚假信号的九大要领

1. 平时话少突然话多 → 有隐瞒
2. 露出畏惧的眼神 → 心虚
3. 把事实和假话混在一起 → 可能不小心说出真话
4. 表现出焦躁不安 → 有不想被揭开的东西
5. 答非所问或对小事夸大其词 → 在回避核心
6. 手脚无意义活动增多 → 内心焦虑
7. 心不在焉或急于结束谈话 → 不感兴趣或有其他顾虑
8. 过分赞同或过分严厉 → 可能在掩饰
9. 接连不断抢先解释 → 提前准备的说辞

### 性格快速判断

| 类型 | 外在特征 | 沟通偏好 |
|------|---------|---------|
| 分裂质（内向型） | 沉默、冷淡、反应慢 | 给空间，不要逼，用文字沟通 |
| 躁郁质（外向型） | 健谈、热情、情绪波动 | 多互动，抓住情绪高点促单 |
| 癫痫质（严谨型） | 一丝不苟、啰嗦、保守 | 提供详细材料，给够时间 |

### 诱导客户说真话的五种技巧

1. **鹦鹉学舌** — 重复对方关键词，促其深入
2. **暧昧法** — 故意说模棱两可的话，等对方自己解释
3. **中断法** — 说到关键处停顿，让对方接话
4. **引例法** — "我有个朋友也遇到类似情况..."（对方往往会代入自己）
5. **反对法** — 故意说反话，刺激对方反驳时暴露真意

## 七、销售心态与自我管理

### 核心信念

1. "这一次的拒绝就是下一次的赞同"
2. "只要决心成功，失败永远不会把我击垮"
3. "弱者任思绪控制行为，强者让行为控制思绪"
4. "取法乎上，得其中也；取法乎中，得其下也"
5. "热情是世界上最大的财富，潜在价值远超金钱与权势"

### 情绪管理（反向行为法）

| 当你感到... | 就做... |
|------------|--------|
| 沮丧 | 引吭高歌 |
| 悲伤 | 开怀大笑 |
| 恐惧 | 勇往直前 |
| 自卑 | 整理仪容 |
| 不安 | 提高嗓音 |
| 穷困 | 想想未来财富 |
| 渺小 | 想想你的目标 |
| 想放弃 | 想想成功后的喜悦 |

### 每日自省清单

每晚问自己三个问题：
1. 今天我发现了什么弱点？
2. 今天我克服了什么情绪？
3. 今天我获得了什么进步？

### 收入恒温器

- 每个销售员心理都有一个"收入恒温器"
- 实际收入不会与心理预期偏离超过10%
- 要提高收入，先升级自我认知中的"应该赚多少"
- 方法：与高收入者交往、阅读成功案例、设定更高目标

## 八、成交时机识别

### 80%法则

80%的购买决定发生在第5次拜访之后。大多数销售在第2次就放弃了。

### 成交信号识别

| 信号类型 | 具体表现 |
|---------|---------|
| 语言信号 | 开始问细节问题（交付时间、付款方式、售后） |
| 行为信号 | 反复查看产品、主动要求试用、介绍其他人参与 |
| 时间信号 | 主动延长谈话时间、约第二次见面 |
| 价格信号 | 开始讨价还价、问有没有优惠 |

### 六种成交手法

1. **假设成交**："下单后我让技术团队明天就上门"
2. **选择成交**："您看是选基础版还是专业版？"
3. **紧迫成交**："这个价格本周截止"
4. **总结成交**：把所有已达成共识列出来，只差签字
5. **试用成交**："先试用一周，不满意随时退"
6. **故事成交**：讲一个类似客户成功购买后获益的故事

### 成交后的关键动作

1. 立刻确认（24小时内发送确认信息）
2. 7天内主动回访使用体验
3. 解决初期小问题（防止退单/退订）
4. 30天后请求转介绍
5. 定期维护关系（节日问候/行业资讯/专属福利）
