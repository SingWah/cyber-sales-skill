## 一、价格异议的5种真实含义

客户说"太贵了"，背后可能是完全不同的情况：

1. **真的买不起** → 预算确实不够 → 提供分期/降级方案，别硬推
2. **在货比三家** → 还没选定你 → 强调你跟别家的核心差异，别打价格战
3. **习惯砍价** → 不砍白不砍 → 守住价格，给个小赠品当台阶
4. **在试你底线** → 想确认自己没吃亏 → 给一个"特殊照顾"的说法让他有面子
5. **需要跟领导交代** → 需要价格理由去汇报 → 帮他准备一份ROI分析，让他好交差

## 二、"让我考虑考虑"的破解

这句话80%的情况意味着"不想买但不好意思直说"。破解思路：

- **锁定具体顾虑**："理解，主要是在考虑哪方面呢？是效果、价格还是其他？"
- **制造紧迫感**（如果是真的）："这个月定的话还能赶上第一批交付，下个月排期就紧了"
- **给退路**："这样，你先不用做决定，我把方案发你看看，有问题随时问我"

## 三、影响力实操技巧

### 互惠
- 免费给点有价值的东西：行业报告、诊断结果、小工具
- 小恩惠换大回报：一杯咖啡可能换一个合同
- 给予要真诚，别给廉价感的东西

### 承诺一致
- 先让客户点头说"是"，再往下推进
- "您觉得这个功能对您有帮助吗？"→ 小认同 → "那整体方案呢？"→ 大认同
- 人一旦公开表态，就有保持一致的压力

### 社会认同
- "你们行业TOP10中有7家在用"——同行选择最有说服力
- 客户推荐/案例是最强的武器，比自己吹100句都管用
- 85%的成交发生在有人说好之后

### 喜好
- 找到跟客户的共同点：同乡、同校、同爱好、同年龄段
- 人们从喜欢的人那里买——先做朋友再做生意

### 权威
- 专业形象 = 产品靠谱：穿着得体、材料专业、数据准确
- 展示资质、奖项、行业认证

### 稀缺
- "仅限本月""只剩3个名额""下一批要等两个月"
- 必须真实——假的被识破一次，信任归零

## 四、产品介绍三段式

介绍产品时，永远按这个顺序说：

1. **这东西是什么**（特征）→ "我们有一套自动化报表系统"
2. **比别人好在哪**（优点）→ "出报表从3天变成3分钟"
3. **对你有什么用**（利益）→ "你每个月省下来的那两个半天，能多跑3个客户"

关键：永远落在"对你有什么用"上。客户不关心你有什么功能，只关心自己能得到什么。

## 五、成交信号识别

客户准备买的时候，会释放这些信号：

| 信号类型 | 具体表现 |
|---------|---------|
| 语言信号 | 开始问交付时间、付款方式、售后这些细节 |
| 行为信号 | 反复看产品、主动要试用、带其他人一起来看 |
| 时间信号 | 主动延长谈话、约下次见面 |
| 价格信号 | 开始砍价、问有没有优惠 |

**出现以上任何信号，立刻切换到成交模式，别再继续推销了。**

## 六、6种成交手法

1. **假设成交**："下单后我让技术团队明天就上门" — 直接当他已经买了
2. **二选一**："您看选基础版还是专业版？" — 不给他"买不买"的选项，只给"买哪个"
3. **紧迫成交**："这个价格本周截止" — 必须是真的
4. **总结成交**：把聊过的共识列一遍，就差他点头
5. **试用成交**："先试一周，不满意随时退" — 降低心理门槛
6. **故事成交**：讲一个跟他情况类似的客户怎么用了之后效果多好

## 七、成交后5步（防退单/促转介绍）

1. 24小时内发确认信息，让他安心
2. 7天内主动回访使用体验，解决初期小问题
3. 有问题立刻处理，别等客户来找你
4. 30天后请求转介绍："身边有朋友也有这个需求的话，帮我推荐一下呗"
5. 定期维护关系：节日问候、行业资讯、专属福利

## 八、客户性格速判

| 类型 | 怎么看出来 | 怎么聊 |
|------|-----------|--------|
| 内向沉默型 | 话少、反应慢、不主动问 | 给空间，别逼，多用文字沟通 |
| 外向热情型 | 健谈、爱聊、情绪波动大 | 多互动，抓住他情绪高点推一把 |
| 严谨细致型 | 问得很细、反复确认、保守 | 给够详细材料，别催他做决定 |

## 九、让客户说真话的5个技巧

1. **鹦鹉学舌** — 重复他说的关键词，他会自己往下深入说
2. **故意含糊** — 说个模棱两可的话，等他自己解释（解释时会暴露真实想法）
3. **关键处停顿** — 说到重点突然不说了，他会接话
4. **拿朋友举例** — "我有个朋友也遇到这个情况..."（他大概率会代入自己）
5. **故意说反话** — 刺激他反驳，反驳时反而说出真话

## 十、销售心态充电

被拒绝时默念：
- "这次不要，下次可能就要了"
- "被客户骂了不代表我差，只代表还没找到他的点"
- "80%的单子是跟了5次以上才成的，大多数同行第2次就放弃了"

情绪低谷时：
- 别窝着，出去走走/打个电话/跟同事聊聊
- 回忆一个成功签单的案例，找回感觉
- 设定今天的小目标："今天只要打通3个电话就算赢"
