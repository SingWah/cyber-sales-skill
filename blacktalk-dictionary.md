# 赛博销售 · 黑话/行话词典

> 销售场景中的"黑话"解码手册，帮助识别客户真实意图

---

## 一、价格与预算类黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "太贵了" | ①习惯性砍价 ②价值感不足 ③真的超预算 | 先问"跟什么比觉得贵？"来区分三种情况 |
| "预算有限" | 可能是真的，也可能是不想花那么多 | "方便告诉我大概预算范围吗？我帮你匹配最合适的方案" |
| "能不能便宜点" | 大概率已决定购买，在争取最后利益 | 不降价，给增值服务："价格确实没办法再降了，但我可以帮你申请XX" |
| "帮我看看有没有优惠" | 给我个心理台阶下/需要向领导交代 | 给一个"限时优惠"或"专属折扣"的说法，让客户有面子 |
| "别家给的更低" | 在压价，可能并没有更低 | "方便说说哪方面吗？有时候看起来便宜，但服务内容差很多" |
| "这个价格我没法跟领导交代" | 需要一个"理由"去汇报 | 提供ROI分析报告/对比表，帮客户做内部汇报材料 |
| "价格可以谈" | 已经决定买了，在争取最后利益 | 守住价格，用小赠品或增值服务替代 |
| "你们最低价是多少" | 在做最终对比，即将下单 | 给出明确底价 + 限时条件，推动决策 |

## 二、决策与时间类黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "我再考虑考虑" | ①不想买 ②有顾虑没说出口 ③需要跟别人商量 | "理解，主要是在考虑哪方面呢？"锁定具体顾虑 |
| "回头再说" | 大概率不会回头 | "好的，那我们约个具体时间再聊？下周二方便吗？" |
| "要走流程" | 可能是真的，也可能是拖延借口 | "流程大概需要多久？中间需要我配合提供什么材料吗？" |
| "领导还没定" | 你可能没接触到真正的决策人 | "方便了解一下这个项目的决策流程吗？需要的话我可以配合做一份给领导的汇报" |
| "先发资料吧" | ①只是收集信息 ②不方便细聊 | 先问需求再发定制化资料，避免进入"方案黑洞" |
| "我现在不方便" | 兴趣不够强/时机不对 | "那您什么时候方便？我到时候再联系您" |
| "年底再定" | 拖延/预算周期/在等其他选项 | "理解。不过现在定的话可以锁定今年价格/提前享受XX" |
| "我们不着急" | 痛点还没到临界点 | 用暗示问题放大痛点："如果这个问题再拖半年，影响会有多大？" |

## 三、竞品对比类黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "你们和XX家比怎么样" | 在做最终选择，接近成交 | 不贬低对手，聚焦自身差异化："我们最大的不同是..." |
| "我朋友也在做这个" | 在用筹码压你，不一定真有 | "那挺好的，方便聊聊他的方案吗？看看我们这边有没有能补充的" |
| "他们家便宜很多" | 在压价，或真的在意价格 | "便宜的方案确实有，但核心区别在于..." |
| "XX家说能做到XX" | 在用竞品的高承诺来压你 | "这个承诺确实吸引人，但实现起来需要XX条件，我们更务实一些..." |
| "你们是不是最大的/最好的" | 我需要安全感/权威背书 | 展示资质、案例、数据，给出确定性 |

## 四、需求与兴趣类黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "我先看看/随便逛逛" | 防御心理强，不想被推销 | "好的，您随意看，有任何问题随时问我" → 用提问切入 |
| "我就了解一下" | 还没到购买阶段，或不想让你知道他在买 | "完全理解。您主要想了解哪方面？我可以针对性地介绍" |
| "这个功能不错" | 找到了兴趣点/热点 | 立刻深挖这个功能的使用场景，锁定热点 |
| "有没有XX功能" | 已有明确需求，在做匹配 | 有就展示，没有就说路线图/替代方案 |
| "我们之前用过类似的" | 有经验，会有对比标准 | "之前使用中有什么满意的和不满意的？" |
| "需求还不确定" | ①真不确定 ②不想告诉你 | 帮客户梳理需求："通常像您这种情况，会关注这几个方面..." |

## 五、售后与关系类黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "上次体验不太好" | 在测试你的态度，或真的犹豫要不要复购 | 真诚道歉 + 具体补救："上次确实是我们的问题，这次我专门..." |
| "最近很忙没顾上" | 优先级不够高/兴趣下降了 | "理解，那我简单跟您同步一下最新进展？只需要3分钟" |
| "帮我推荐一下" | 信任你了，愿意做转介绍 | 立刻提供具体转介绍话术 + 感谢回馈 |
| "你们售后怎么样" | 担心买了之后出问题没人管 | 展示售后体系、响应时间、案例 |
| "有问题找谁" | 担心被"踢皮球" | 给出具体联系人 + 承诺响应时间 |

## 六、微信/线上沟通黑话

| 客户的表现 | 真实含义 | 应对策略 |
|-----------|---------|---------|
| 已读不回 | ①不感兴趣 ②太忙忘了 ③在犹豫 | 3天后发一条有价值的内容（不是"在吗"） |
| 回复"嗯""好""收到" | 敷衍/不想继续聊 | 换话题或用提问重新引起兴趣 |
| 秒回+问很多细节 | 高意向客户，正在评估 | 耐心解答，趁热打铁约面谈/电话 |
| 发语音而不是打字 | 愿意花时间沟通，信任度较高 | 也用语音回复，拉近距离 |
| 朋友圈互动多 | 在维护关系/有潜在需求 | 积极互动，择机私聊 |
| 突然删好友/拉黑 | 之前跟进太紧/价值感不足 | 反思跟进方式，必要时换号重新建联 |

## 七、B2B/大客户场景黑话

| 客户说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "我们在做选型" | 已经立项了，在做最终比较 | 了解选型标准，针对性准备材料 |
| "帮我出个方案" | 可能在收集方案向上汇报，不一定选你 | 确认需求深度，避免做"陪跑方案" |
| "领导想了解一下" | 决策层开始关注，关键节点 | 准备高层视角的精简版材料 |
| "你们有案例吗" | 需要安全感，需要可参照的成功故事 | 给出同行业/同规模的成功案例 |
| "这个需要招标" | 可能走流程，也可能已有意向厂商 | "招标大概什么时间？我们可以提前配合准备" |
| "预算还没批" | 项目优先级不够/在等更好的时机 | 帮客户做内部ROI论证 |
| "你们有没有XX资质" | 在做合规审查/准入评估 | 提供资质文件 + 解释资质背后的能力 |

## 八、谈判场景黑话

| 对方说的 | 真实含义 | 应对策略 |
|---------|---------|---------|
| "这是我们的底线了" | 通常不是底线，还有空间 | "理解，那我们看看在XX方面能不能灵活一些" |
| "我做不了主" | ①真的需要上级 ②在拖延/施压 | "方便跟决策者一起聊吗？我来做汇报" |
| "别人给的条件更好" | 在施压让你让步 | "方便说说哪方面吗？条件不同往往是因为方案不同" |
| "今天能定的话..." | 在给你机会，但也可能是在施压 | 确认对方诚意，给出有条件的让步 |
| "我让步已经很大了" | 接近真实底线了 | 换非价格维度谈判（付款周期/交付时间/增值服务） |
| "不谈了" | ①真的要放弃 ②在试探你的底线 | 冷静应对，给台阶："那我们各退一步？" |

---

## 使用原则

1. **不要仅凭一句话判断** — 结合上下文、语气、历史互动综合判断
2. **解码是为了更好服务，不是操控** — 目的是理解真实需求，提供更好的解决方案
3. **验证你的判断** — 解码后用提问验证："您是不是在担心XX？"
4. **保持真诚** — 即使看穿了客户的"套路"，也不要揭穿，而是顺势引导
